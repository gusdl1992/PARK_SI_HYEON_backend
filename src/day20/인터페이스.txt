    1. 두 객체를 연결하는 역할.
    2. 인터페이스 구현 , 인터페이스 활용
    3. 객체 생성 불가. ( 구현 글래스 필요 -> 재정의 메소드 필요 )
        1. 선언 interface 인터페이스명 {}
        2. 구현클래스 : class 클래스명 implements 인터페이스 {}
    4. 인터페이스 멤버
        1. 상수 필드 ( public static final -> 생략가능)  , 인스턴스 필드 불가.
        2. 추상 메소드 ( public abstract -> 생략가능. ) , 선언부 작성 , 구현부{} 작성안함. ( 구현객체 필요 )
        3. 디폴트 메소드( default ) , 선언부 작성 , 구현부 작성 , 오버라이딩 선택 ( 구현객체 필요 )
        4. 정적 메소드 ( public static ) , 선언부 작성 , 구현부 작성 ( 구현객체 필요 없음 )

            private 메소드                 : 구현객체가 필요한 메소드
            private 정적(static) 메소드     :구현객체가 필요 없는 메소드
